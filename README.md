**Данные о системе**

1. Решение было запущено на Ubuntu-20.04 под Windows 10
2. Видеокарта: Nvidia GeForce RTX 3090Ti
3. Python: 3.11.7

Для установки всех зависимостей выполните команду: pip install -r requirements.txt --force. Убедитесь, что у вас установлен jupyter notebook.

**Создание базы данных**
1. Распакуйте все pdf-инструкции и excel-файл с запросами
2. Откройте файл db_creation.ipynb
3. Укажите нужные пути в переменной inst_path и строке questions_history = pd.read_excel('Вопросы и ответы.xlsx')
4. Запустите файл. Будут созданы 3 .pkl файла для БЗ

**Запуск сервера с LLM**
1. Убедитесь, что у Вас не менее 16 Гб видеопамяти с поддержкой CUDA
2. Перенесите созданные файлы бз в папку с сервером и выполните команду python3 server.py
3. Сервер запущен и готов обрабатывать запросы.
4. **ОЧЕНЬ ВАЖНО**: не выполняйте более одного запроса за один раз, иначе сервер упадет, тк предназначен для демонстрации, а не для прома
